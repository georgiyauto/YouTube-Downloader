# YouTube Downloader GUI

Приложение для скачивания видео с YouTube с графическим интерфейсом.

## Функции

- Ввод URL видео
- Получение доступных качеств видео
- Выбор качества из выпадающего списка
- Галочки для выбора скачивания видео/аудио
- Progress bar для отображения прогресса скачивания
- Автоматическое открытие папки с загрузками после завершения
- Сохранение в папку `Downloads/YouTube Downloads`

## Установка и запуск

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Запуск приложения

```bash
python youtube_downloader_gui.py
```

### 3. Создание exe файла

```bash
pip install pyinstaller
python build_exe.py
```

После сборки exe файл будет находиться в папке `dist`.

## Использование

1. Введите URL видео в поле ввода
2. Нажмите "Получить доступные качества"
3. Выберите желаемое качество из списка
4. Отметьте галочками что скачать (видео, аудио)
5. Нажмите "Скачать"
6. Дождитесь завершения и откройте папку с загрузками

## Системные требования

- Python 3.7+
- Windows/macOS/Linux
- Интернет соединение
